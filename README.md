# Mini Project kafka-connect-to-bigquery


## Overview
Create pipline kafka-connect-to-bigquery
- Change Data Capture (CDC)
- Streams changes through Kafka and writes them to a target database in real time
- cleansing data using Apache Spark
- Export data to BigQuery

## Tools
- Apache Kafka
- Apache Spark
- Google Cloud Platform (GCP)

## Features
- streming data from Kafka
- Change Data Capture (CDC)
- Export data real time
- Data cleansing using Apache Spark
- Data storage in Google Cloud Storage
- Data storage in BigQuery
